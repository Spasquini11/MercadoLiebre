Para el boton de enviar
<button type = "reset">Borrar</button>
<button type = "submit">Enviar</button> Va al action de la etiqueta que pusiste en el <form>
<button type = "button"></button> +JS

atributo name, para los campos de un formulario.
atributo value, para los campos que no permite texto por los usuarios. <options value = 'Arg'> Argentina </option>
atributo required, hace el campo obligatorio. <input type = "text" name = "NombreDeUsuario required>
atributo placeholder, texto de ayuda. <input type = "text" name = "NombreDeUsuario placeholder = 'Ej: Santiago' required>
