Aparte del posionamiento flex box, tenemos otras cosas. Fijar, posicionar atras o adelante...

**POSITIONS**

Position no cancela flebox

position: static; (flujo normal).

position: relative (cambiar de lugar a su posicion original).
top: 50px;
left: 100px;

position: absolute. (Se desplaza h y v segun los costados del body)
bottom:50px;
right: 50px;
Aca, de abajo a la derecha. 
La caja deja su espacio original y la de abajo ocupa su lugar. 
Si las cajas estan dentro de un contenedor la referencia es el body, salvo que al contenedor le pongas relative y asi se posiciona en funcion a su caja contendora.

position: fixed; (para fijar al scrollear).


**Z-INEDX**

La profundidad... El valor asignado puede ser positivo o negativo.
+ valor + arriba.
Si hay dos iguales con igual z-index, el que esta en el HTML ultimo va por encima.
Solo funciona en elementos posicionados (todas menos estatica).


**Transiciones** solo para con hover.

.mi-elemento{
    transition: border-radius 450ms.
}

.mi-elemento: hover{ 
    border-radius: 50%;
} 


**Animaciones** fuera de hover. Con Frames o fotogramas.

@keyframes crecimiento{

    0%{
        background-color: red;
    }

    100%{
        background-color: green;
        height; 200px;
        width; 200px;
        opacity: 0.5;
    }
}

Ahora la aplicas sobre el elemento a afectar.

.mi-elemento{
    height; 200px;
    width; 200px;
    border-radius: 100%;
    animation-name: creciendo;             
    animation-duration: 1000ms;
    animation-iteration-count: infinite;                //cant de veces a reproducir.
    animation-direction: alternate;                     //al final, se genera como que vuelve.
}
